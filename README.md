# Compound Interest Calculator

A command line program written in C that calculates compound interest based on user input.

The program allows the user to enter the principal amount, interest rate, compounding frequency and number of years.  
It then calculates the final amount using the compound interest formula.

The result can also be converted into Serbian dinars depending on the chosen currency.

## Features

- Calculate compound interest
- Support for multiple compounding periods per year
- Currency selection
- Conversion to Serbian dinars
- Simple command line interface

## Compound Interest Formula

The program uses the standard compound interest formula:

A = P (1 + r / n)^(n × t)

Where:

- **P** = principal amount  
- **r** = annual interest rate  
- **n** = number of times interest is compounded per year  
- **t** = number of years  
- **A** = final amount
